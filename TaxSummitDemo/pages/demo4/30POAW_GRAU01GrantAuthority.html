Instructions

All tax types

This includes all taxes and fees the Tax Department is responsible for collecting and enforcing, other than for estate tax.

Other tax types

You will be able to select a specific tax type if you select this box.

My collection case

If you are granting authority for this collection case, check the box for My collection case. This includes any open unpaid tax debts that are under collection.

My audit cases

If you are granting authority for specific audit cases, check the box for My audit cases and select the audit case IDs. You can grant authority for up to 20 audit cases. If you don't see your case number listed, you can remove one and add another.

My bills

If you are granting authority for specific bills, check the box for My bills and select the assessment IDs. You can grant authority for up to 20 bills. If you don't see your ID number listed, you can remove one and add another.

Add limitation on authority

If you want to limit your representative’s authority, explain the limitation. For example, you can limit your representative’s authority to only receive confidential information, but make no binding decisions for you.
