Special condition

Please select the special condition that may apply to your filing period. (This doesn't apply to everyone.)

  • Out of country (E3) - Use this code if you qualify for an automatic two-month extension of time to file your federal and New York State income tax returns because you are out of the country and you need an additional four months to file. This code typically applies to U.S. citizens abroad.
  • Nonresident alien (E4) - Use this code if you are a U.S. nonresident alien for federal income tax purposes, or a nonresident alien estate or trust and you qualify to file your federal and New York State income tax returns on June 15, 2017, and you need an additional six months (five months for nonresident alien estate or trust) to file. This code typically applies to U.S. citizens abroad.
  • Electing large partnership (B5) - Use this code if you are an electing large partnership and need an additional six months to file or qualify for an additional six months to file.
  • Death of Spouse (D9) - Use this code if you qualify for an automatic Ninety-day extension of time to file your return because your spouse died within thirty days before the due date of your return.
